# ADR-000:采纳 Governance Kit 作为本项目治理基座

- 背景:AI 主导执行的长期项目,需对抗五个结构问题(S1 失忆 / S2 意图不可传心 / S3 宣称不可自证 / S4 不可逆动作 / S5 注意力预算)。
- 决定:按 ai-governance-playbook v2 安装(载体阶梯 + 事故驱动立法 + 双侧编译 + 单一心跳);Day-1 只建结构底座。
- 否决了:预防式全量规则手册(无校准数据、稀释注意力);逐项人工审批(监督成本随吞吐线性增长)。
- 后果与回退条件:接受「每类错误交一次学费」;若同类错误二犯率持续 >0 且载体升级失效,重审基座。安装环境:{Claude Code / Codex / 裸 agent,由安装 AI 填写},关键载体降级项:{无,或列出}。
