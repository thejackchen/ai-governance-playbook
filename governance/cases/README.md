# 判例库（品味的判例法）

> 不要预填虚构示例；真实判例（含母版反哺）随时收录。负责人每次纠正或拍板，执行者当场代笔一事一文并登记进下方索引。新项目接入前先通读——历次教训拿来即用。

判例格式（六字段，一事一文）：

- **来源**：{项目 · 日期 · 触发方式（负责人当场纠正/裁决/上游反哺）}
- **场景**：
- **执行者做法**：
- **负责人要的做法**：
- **可泛化边界**：{若提炼得出；没有就空着}
- **载体**：{该教训已下沉/待下沉到哪个载体；描述性写法，不写指向个人机器的绝对路径}
